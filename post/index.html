2018

@Nemocdz · Jan 1, 2019 · 2 min read

今年对我来说是特别的一年，22 岁，从大学顺利毕业，开始工作。 大学 自从去年 10 月份签了 Offer 后，没打算去实习，就开始享受大学的最后半年多时光了。其实对于我的大学，我的情感是很复杂的，一方面，在知识学术上，大学其实并没有给我特别大的帮助，除了本身我的专业（网络工程）的专业课我都不太感兴趣之外（大学四年基本没去上过课），另一方面，我的大学又给我了很多无可代替的东西，三个可爱无比的室友，一群摄影工作室的朋友，一群写代码的朋友，图书馆各种最新的技术书籍，成都的美食和文化，四川的风景，985 的学历，还有恋爱经历等等。 大学还有四门印象深刻的课，《通信原理》，《大学物理上》，《大学物理下》，《DSP（数字信号处理）》。直到大四下的清考，我才把它们修过……大物下还全靠老师抬一手。再次确定了我这个人，不感兴趣的东西很难愿意去学。 毕设的课题本来选了《UDP 不丢包的方案》的，我当时想难道这个导师有什么特别的方案？结果和导师聊了下发现他其实也没有具体想法……然后就换成了给他们教研室的光纤卡写个 Android 的测试软件，就当写外包了，顺便了解下 Android 开发也挺好的。 清考过了，论文交了，毕业照拍了，毕业旅游去了，毕业典礼完了，各种大小团体聚完餐，就毕业了。我的宿舍，一个保研搞机器学习去了，一个回老家当公务员，一个去重庆做本专业的开发去了，而我到深圳当了一个开发仔，感觉大学的人儿，轨迹都截然不同，分散在各个地方，见面其实是很难的。 总的来说，我还是十分感激和怀念我的大学的。 工作 下半年来到深圳，入职鹅厂，正式成为一名程序员。首先享受了一周带薪的封培，认识了一群朋友圈点赞的朋友，上了公司文化的一些课。鹅厂的工作环境还是不错的，iMac 5K + 13 寸 MacbookPro + 1080p 显示器（这个有点…），早餐免费，晚餐基本等于免费（每天8 点后打卡有夜宵劵），10 点后打车免费，每个月点卷福利，上下班班车等等。而我在的科兴园区，早餐有稻香吃早茶，晚餐可以刷麦当劳肯德基赛百味，真的胖。就是电梯实在是堵值得吐槽。不过内部开发环境不太友好，抓包，邮件代理，命令行代理都比较麻烦，而且技术用得也比较陈旧（我们组还好），事业群之间壁垒有些重。当然，规范，专业，开放，关怀，这些其实是鹅厂很不错的地方。 快一年没写过 iOS 了（自从去年 8 月份实习结束后），花了点时间重新学了一下。然后我们事业群毕业生还要完成一个 Mini Game 的项目，随机组队，只有一周时间脱产，所以基本就是周末加班两天，当然我们组两个程序以前都没做过游戏开发，所以从头学习了 Unity + C#，总算完成了一个 2D 横版过关游戏《Rebecca》，虽然没拿奖（评委嫌 2D 横版过关没有创意），但认识的 Mini Game 的小伙伴还是很开心的。 后面就正式进入了糊 UI 的工作，由于我们组比较开放，在别的组还在 SVN + 原生的时候，我们组和正常的 8012 年互联网公司一样，用上了 Git + React Native，Android 用上了 Kotlin，所以我就把 Swift 引入了我们组里，不过我之前从来没有写过 Swift，所以还在持续学习中。后面写了一段时间的 React Native，也是没有接触过的，虽然 Javascript 以前写过，但是写得很少，React 没有接触过，感觉还是十分新奇的。还给组里写了一些组件和引入了 Carthage，不过 Leader 说我的技术影响力还是不够，就是说很多东西引入后没有让组里大家用起来。这个我觉得是我需要提高的，怎么说服老员工一起用一些前沿的技术。
